安卓手机如何访问电脑上的localhost网页 #

把手机 和 电脑 连接同一个局域网中，让后把电脑上的localhost URL ，
修改成 电脑对应的 IP 地址 。
比如说 url 为 localhost：80/welcome.html 电脑IP 为 192.168.0.12
则修改为 192.168.0.12：80/welcome.html.
