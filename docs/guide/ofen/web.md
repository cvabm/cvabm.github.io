# web 前端

## http-server

```
npm i -g http-server
在要成为服务器的目录下运行如下命令
http-server
若要禁用缓存，请使用如下命令运行
http-server -c-1
运行后在浏览器输入http://localhost:8080/或者http://127.0.0.1:8080
```
