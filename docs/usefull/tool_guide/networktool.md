## 模拟弱网环境clumsy
<https://jagt.github.io/clumsy/cn/index.html>
```
延迟(Lag)，把数据包缓存一段时间后再发出，这样能够模拟网络延迟的状况。
掉包(Drop)，随机丢弃一些数据。
节流(Throttle)，把一小段时间内的数据拦截下来后再在之后的同一时间一同发出去。
重发(Duplicate)，随机复制一些数据并与其本身一同发送。
乱序(Out of order)，打乱数据包发送的顺序。
篡改(Tamper)，随机修改小部分的包裹内容。
```
